# AI/ML Job Salary Prediction

This project predicts the average salary of AI/ML job listings using machine learning techniques.

## Dataset
Kaggle AI/ML Job Market Dataset

## Models Used
- Linear Regression
- Random Forest Regressor
- Tuned Random Forest Regressor

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Tools & Libraries
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Streamlit

## Objective
To analyze AI/ML job market data and predict average salary using regression models.
